339/* RSA Keys */
340/*
341 * n, e, d are the usual public and private key components
342 *
343 * rsa-num is the number of factors, including p and q
344 * rsa-factor is used for each factor: p, q, r_i (i = 3, ...)
345 * rsa-exponent is used for each exponent: dP, dQ, d_i (i = 3, ...)
346 * rsa-coefficient is used for each coefficient: qInv, t_i (i = 3, ...)
347 *
348 * The number of rsa-factor items must be equal to the number of rsa-exponent
349 * items, and the number of rsa-coefficients must be one less.
