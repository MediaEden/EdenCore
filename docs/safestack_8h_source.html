181/*-
182 * Strings are special: normally an lhash entry will point to a single
183 * (somewhat) mutable object. In the case of strings:
184 *
185 * a) Instead of a single char, there is an array of chars, NUL-terminated.
186 * b) The string may have be immutable.
187 *
188 * So, they need their own declarations. Especially important for
189 * type-checking tools, such as Deputy.
190 *
191 * In practice, however, it appears to be hard to have a const
192 * string. For now, I'm settling for dealing with the fact it is a
193 * string at all.
